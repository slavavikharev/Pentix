Игра Pentix

Суть игры: 
    набрать как можно больше очков путем расставления падающих блоков.
    Очки начисляются за упавший блок (сколько клеток - столько очков)
    и за удаленный ряд (+"ширина поля" очков)

3 уровня игры:
    Уровни отличаются скоростью падения блоков.

Управление:
    Стрелка влево - сдвинуть падающий блок влево
    Стрелка вправо - сдвинуть падающий блок вправо
    Стрелка вниз - ускорить падение блока
    Стрелка вверх - повернуть блок на 90 градусов по часовой стрелке

Запуск игры:
    python3 pentix.py
